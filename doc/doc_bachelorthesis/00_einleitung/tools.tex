\section{Tools}
\label{sec:einleitung.Tools}

Für diese Arbeit verwendeten wir folgende Tools:
\begin{itemize}
\item Eclipse für die Java-Entwicklung (\url{http://www.eclipse.org})
\item ANT für die Build-Automatisierung (\url{http://ant.apache.org})
\item Git (\url{http://git-scm.com}) für die Versionskontrolle, mit einem zentralen Repository für die einfachere Zusammenarbeit auf GitHub (\url{http://www.github.com})
\item TeXnicCenter (\url{http://www.texniccenter.org}) und MiKTeX (\url{http://miktex.org}) für die Dokumentation in \LaTeX
\item GanttProject (\url{http://www.ganttproject.biz}) für den Zeitplan
\item Visual Paradigm for UML (\url{http://www.visual-paradigm.com/product/vpuml/}) für die UML-Diagramme
\end{itemize}
