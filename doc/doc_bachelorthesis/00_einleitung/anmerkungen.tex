\section{Anmerkungen zur Dokumentation}
\label{sec:einleitung.Anmerkungen}
Hier einige Anmerkungen zur Dokumentation:

\paragraph{Diagramme}
Sämtliche \gls{UML}-Diagramme stellen jeweils eine vereinfachte Sicht dar und erheben keinen Anspruch auf Vollständigkeit. Bei den Diagrammen, die einzelne Packages darstellen, sind jeweils Klassen oder Interfaces, die aus einem anderen Package stammen, farblich hervorgehoben.

\paragraph{Listings}
Die meisten Listings sind direkt aus dem Code unseres Bots kopiert. Der Lesbarkeit halber wurden sie jedoch mehr oder weniger stark gekürzt und einzelne Abschnitte wurden durch Pseudocode ersetzt.
